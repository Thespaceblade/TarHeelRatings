View Project Read Privacy Policy

What it does

TarHeelRatings helps students make faster registration decisions by showing useful professor information where course choices already happen.

Inline professor insights

Adds rating indicators next to instructor names and shows a rich hover card with overall rating, difficulty, rating count, would-take-again percentage, and latest review timing when available.

Built for real registration flow

Works in class search and shopping cart views, including support for multiple instructors and the updated reskinned card/list shopping cart experience.

Fast local caching

Uses local browser storage to cache recent lookups temporarily, which speeds up repeat page loads and reduces unnecessary requests for the same instructors.

How it works

The extension runs only on supported UNC ConnectCarolina pages, reads instructor names from the page, requests public rating data from RateMyProfessors, and displays the results inside the page interface.

Why students use it

  • See instructor context without opening separate tabs.
  • Compare options faster during high-pressure registration windows.
  • Make better-informed choices from the same page where enrollment happens.

Important notes

  • TarHeelRatings is an independent project and is not affiliated with UNC Chapel Hill or RateMyProfessors.
  • Not every instructor has a matching RateMyProfessors profile.
  • Ratings may be sparse or imperfect, and should be treated as one input among many.

Overview

TarHeelRatings is a browser extension designed to display RateMyProfessors information directly within UNC Chapel Hill's ConnectCarolina registration experience. This privacy policy explains what the extension processes, what it stores, and how that information is used.

What TarHeelRatings does

  • Reads instructor names and related course page content on supported ConnectCarolina pages.
  • Requests publicly available professor rating data from RateMyProfessors.
  • Displays rating summaries and hover details in the page interface.
  • Caches recent lookup results locally in the browser to improve speed and reduce repeated requests.

Information processed by the extension

TarHeelRatings may process website content from supported ConnectCarolina pages, such as instructor names and related class listing information, only to match instructors with publicly available RateMyProfessors data and display those results to the user.

TarHeelRatings does not request or intentionally collect personal account credentials, financial information, health information, private communications, or unrelated browsing history.

Local storage usage

The extension uses Chrome local storage to store temporary cached professor rating lookup results. This cache improves performance, helps repeated pages load faster, and reduces unnecessary repeat requests for the same instructor data.

Cached data is used only for the extension's functionality and remains stored locally in the browser environment.

Third-party requests

TarHeelRatings sends requests to RateMyProfessors to retrieve publicly available rating information for instructors shown on supported ConnectCarolina pages. The extension does not download or execute remote JavaScript or WebAssembly as part of this process.

Data use commitments

  • TarHeelRatings does not sell user data.
  • TarHeelRatings does not use or transfer user data for advertising.
  • TarHeelRatings does not use or transfer user data for purposes unrelated to its single purpose.
  • TarHeelRatings does not use or transfer user data to determine creditworthiness or for lending purposes.

Data retention

Cached lookup results are retained locally for a limited period to support the extension's performance goals. They may be overwritten or removed automatically over time as the cache rotates.

Children and educational use

TarHeelRatings is intended for students using UNC Chapel Hill's registration system. The extension is not specifically directed to children.

Policy updates

This privacy policy may be updated from time to time to reflect product changes, platform requirements, or legal updates. Any revised version will be posted on this page.
