Features

Made to save clicks, not reinvent registration.

The goal is simple: keep professor context inside the pages you already use, especially class search and shopping cart.

Inline professor ratings

Instructor names get a rating marker right in the page, plus a hover summary with overall rating, difficulty, rating count, and would-take-again data when available.

Works in actual UNC flows

Built around the pages UNC students actually touch: class search, shopping cart, and multi-instructor course listings.

Local cache for speed

Recent lookups are cached in the browser so the same names do not need to be fetched over and over again.

How it works

Reads the instructor names, fetches public RMP data, and drops it back into the page.

It only runs on supported ConnectCarolina pages. It does not touch your account settings or try to do anything outside the extension's one job.

Why people use it

  • Less tab switching during registration.
  • Faster comparison between sections and instructors.
  • A quick reality check before locking in a schedule.

Good to know

  • Some professors will not have a clean match on RateMyProfessors.
  • Low rating counts can be noisy.
  • RMP data should be used as context, not as the whole decision.

Overview

TarHeelRatings is a browser extension that displays RateMyProfessors information inside UNC Chapel Hill's ConnectCarolina registration experience. This policy explains what the extension processes, what it stores locally, and how that information is used.

What the extension does

  • Reads instructor names and related class listing content on supported ConnectCarolina pages.
  • Requests publicly available professor rating data from RateMyProfessors.
  • Displays rating summaries and hover details in the page interface.
  • Caches recent lookup results locally to make repeat loads faster.

What information is processed

TarHeelRatings may process website content from supported ConnectCarolina pages, such as instructor names and related class information, only for the purpose of matching instructors with publicly available RateMyProfessors data.

TarHeelRatings does not intentionally collect account credentials, payment details, health information, private communications, or unrelated browsing history.

Local storage

The extension uses Chrome local storage to keep a temporary cache of professor rating lookup results. This improves performance and avoids unnecessary repeat requests for the same instructor data.

Third-party requests

TarHeelRatings sends requests to RateMyProfessors to retrieve publicly available rating information for instructors shown on supported pages. The extension does not download or execute remote JavaScript or WebAssembly.

Data use commitments

  • TarHeelRatings does not sell user data.
  • TarHeelRatings does not use or transfer user data for advertising.
  • TarHeelRatings does not use or transfer user data for purposes unrelated to its single purpose.
  • TarHeelRatings does not use or transfer user data to determine creditworthiness or for lending purposes.

Retention

Cached lookup results are retained locally for a limited time and may be overwritten or removed automatically as the cache rotates.

Updates

This policy may be updated if the extension changes, platform requirements change, or legal requirements change. Any updated version will be posted on this page.
